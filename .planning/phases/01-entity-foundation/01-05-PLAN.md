### 6. Configure JaCoCo Coverage

Add JaCoCo plugin to pom.xml with configuration:
- **Goal:** report
- **DataFile:** ${project.build.directory}/jacoco.exec
- **OutputDirectory:** ${project.reporting.outputDirectory}/jacoco
- **Excludes:** 
  - *Test* classes
  - Domain entity classes (EntityType, Property, EntityInstance)
  - DTO classes
  - Configuration classes
  - Exception classes
- **Minimum coverage:**
  - INSTRUCTION: 80%
  - BRANCH: 70%
  - CLASS: 80%

---

### 7. Verify Coverage

Run tests with coverage report:
```bash
mvn clean verify
```

Check coverage report:
- Target: `bedomain/target/site/jacoco/index.html`
- Verify: 80%+ line coverage
