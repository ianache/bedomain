<tasks>

<task type="auto">
  <name>Task 1: Add EventPublisher to EntityInstanceService</name>
  <files>bedomain/src/main/java/com/bedomain/service/EntityInstanceService.java</files>
  <action>
Modify EntityInstanceService.java:
1. Add EventPublisher as a dependency (inject via constructor with @RequiredArgsConstructor)
2. In create() method: After entityInstanceRepository.save(), call eventPublisher.publishEntityCreated(entityInstance)
3. In update() method: After entityInstanceRepository.save(), call eventPublisher.publishEntityUpdated(entityInstance)

IMPORTANT: Event publishing must happen AFTER the save completes (after @Transactional commits). Do NOT call eventPublisher inside the transactional method - call it after the save returns but before returning the response. This prevents publishing events for entities that may be rolled back.

Note: For update, we need previousState for the event. Since EntityInstance has currentState field, and we're updating it, we can pass null for previousState in the update event (EVNT-03 only requires entity update, not state change). For state changes, StateTransitionService handles it.
  </action>
  <verify>
cd bedomain && mvn compile -q
  </verify>
  <done>EntityInstanceService injects EventPublisher, publishes events on create and update</done>
</task>
